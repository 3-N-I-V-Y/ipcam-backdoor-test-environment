통합 제어 시스템(Control-plane)

콘솔에서 정상적인 제어 통신(Control-plane)이 가능합니다. 여기에는 카메라 연동, RTSP 스트리밍 전송, 작업 폴링, 비콘(생존 신호) 보고 및 관리자 제어 기능이 포함됩니다.

Normal path Camera -> Control Server + MediaMTX -> NVR recorder

Use this baseline before adding rogue control, exfiltration, or botnet-like traffic.
